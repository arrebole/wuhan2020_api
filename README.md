### 提交结构

> 接口POST http://wuhan2020.muxxs.com/api/add

文章

| 字段          | 说明     |
| ------------- | -------- |
| announce_type | 默认 0 |
| city          | 市级名   |
| content       | 主要内容 |
| link          | 原文链接 |
| links_to_pic  | 图片链接 |
| province      | 省名     |
| publish_date  | 发布日期 |
| publish_time  | 发布时间 |
| title         | 标题     |
| uploader      | 提交者   |

>GET http://wuhan2020.muxxs.com/api/read?city=<城市名>
